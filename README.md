# Web_Vue_clock
網頁計時器製作

# 範例
https://johnworker.github.io/Web_Vue_clock/

# CDN

```
    <script src="https://unpkg.com/vue@next"></script>
    <script src="https://johnworker.github.io/Web_Vue_clock/main.js"></script>
```

# 說明

想要顯示時鐘效果元素添加 id 與資料

```
<div id="clock">
    {{ h }} : {{ m }} : {{ s }}
</div>

```
